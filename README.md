# Month 1 - Python Basics and Data Manipulation

## Project Overview

This repository contains the projects and exercises completed during Month 1 of my Data Science Internship. The focus of this month was to build a strong foundation in Python programming, data manipulation, and data visualization.

The project covers Python fundamentals, NumPy, Pandas, Matplotlib, and Seaborn through practical exercises and mini-projects.
